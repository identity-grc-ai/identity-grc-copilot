# Demo Script – Identity GRC Copilot

## Scenario: Identity Security Risk Analysis

### Step 1 – Overview

User:
"What are the main identity security risks in our environment?"

Expected Agent Response:
- Risk Level: HIGH
- MFA coverage gaps
- Legacy authentication enabled
- Privileged accounts risk

---

### Step 2 – MFA Analysis

User:
"Do we have any gaps in MFA enforcement?"

Expected Agent Response:
- MFA is only 72%
- 3,200 users not protected
- High risk identified

---

### Step 3 – Impact Simulation

User:
"What would happen if we enforced MFA for all users?"

Expected Agent Response:
- Risk reduced significantly
- Security improved by ~60–80%
- Low user impact
