# Identity GRC Copilot

## Overview
An AI-powered Enterprise Agent built with Microsoft Copilot Studio that analyzes identity security posture and provides actionable recommendations.

## Problem
Organizations lack real-time visibility into identity security risks such as MFA gaps, misconfigured Conditional Access policies, and privileged access exposure.

## Solution
A Copilot agent that:
- Analyzes identity security data
- Detects risks and misconfigurations
- Provides remediation recommendations
- Estimates security impact

## Features
- Identity risk analysis
- MFA compliance checks
- Conditional Access evaluation
- Risk scoring
- Actionable recommendations
